# chatgptpanel

Access OpenAI ChatGPT in a mini window, perfect for multitasking devs to stay connected without leaving their workspace.

## Development

### Prerequisites
- Node.js
- PowerShell 7+

### Setup
```powershell
npm install
```

### Testing
```powershell
npm test
```

### Formatting
```powershell
npm run format:all
```
